<h1>Calculadora em Dart</h1>

Este projeto é uma calculadora básica feita em Dart.
O objetivo é apenas aprender e praticar a sintaxe da linguagem, trabalhando com entrada de dados, funções e operações matemáticas.

Ela permite fazer: soma, subtração, multiplicação e divisão pelo terminal.

Para usar, execute o código e siga as instruções que aparecem na tela.
